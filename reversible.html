    PDF versions of the encoding charts and other helpful materials

    Please feel free to test the tool yourself. If you have questions or comments, contact Prof. Walter Andrews at walter@uw.edu

    To test the experimental Reversible Transcription Tool, do as follows:

    1. Go to the "helpful materials" URL above, open the reversible transcription samples file and copy the first poem (omitting the title in square brackets).
    2. Go the the RT Tool Intake Page at the following URL
      http://courses.washington.edu/otap/reverse/intake/reverse_tool_intake.html
    3. Paste the RT Sample into the intake box and click on Ottoman Turkish below.
    4. You can then delete the Arabic script text, copy the RT text into the box again and click on the Ottoman Latin button below.

    Then you can use the charts and make your own reversible transcription of some Ottoman Text. It will seem complex and daunting at first but if you have gotten this far you have already learned the Arabic alphabet as used by the Ottomans, Ottoman Turkish, Arabic, Persian, and probably other languages as well. You will learn this too, and quickly!
